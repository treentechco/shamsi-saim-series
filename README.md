## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Customization

- Replace placeholder images in `app/page.jsx` with your own wedding photos
- Update contact links (Instagram, WhatsApp, Email) in the component
- Modify colors and styling in `tailwind.config.js`

## Build for Production

```bash
npm run build
```
